# x/twitter clone

the goal of the project is to practice backend in GO while coding smth relatively interesting

## features v1

- [ ] authentication
- [ ] basic profile CRUD
- [ ] search users
- [ ] follow/unfollow
- [ ] post a tweet (text only)
- [ ] see feed (tweets from ppl u follow)
- [ ] see user's profile and their tweets

## technologies

- net/http for routing
- PostgreSQL (database/sql + pgx)
